Programming Language

A programming language is a formal set of rules and instructions that lets humans communicate tasks and commands to a computer, enabling the creation of software, apps, and scripts

Programming Languages are two types:

  1. Interpreted Languages

    2. Interpreted languages are programming languages in which the source code is executed line by line or instruction by instruction directly by an interpreter program, without being first compiled into machine code. This means the computer runs the code on the fly, translating each instruction as it is encountered during runtime.

    1. Python
    2. JavaScript
    3. Ruby

  2. Compiled Languages

    3. A compiled language is a programming language where the source code you write is first translated by a compiler into machine code (binary instructions) that the computer's processor can execute directly. This compilation step happens before the program is run, creating an executable file independent of the source code.

    • C
    • C++
    • Go

Popular Languages

  1. Python
    • It is interpreted, meaning code runs line-by-line without needing separate compilation.
    • It supports multiple programming styles: procedural, object-oriented, and functional.
    • It uses indentation (spaces/tabs) to define blocks of code instead of braces.
  2. Java
    • Object-Oriented – Uses classes, objects, and OOP principles.
    • Platform Independent – “Write Once, Run Anywhere” via JVM.
    • Simple Syntax – Easier to learn; avoids complex features like pointers.
  3. C++
    • Object-Oriented – Supports classes, objects, inheritance, polymorphism, etc.
    • High & Low-Level Features – Combines advanced abstraction with direct hardware access
    • Compiled Language – Fast execution and efficient resource use.
